## 🚀 Featured Projects

### 🍽️ [Nexus-Culinary](https://github.com/alludhanush777-gif/Nexus-Culinary)
A food ordering web platform built as a demo for college canteen use — makes browsing and ordering food simple, with easy-to-follow options and meal ideas for beginners and regular users.
- **Stack:** JavaScript, HTML/CSS, MongoDB
- **Live Demo:** [nexus-culinary.vercel.app](https://nexus-culinary.vercel.app/)
- **Challenge:** Designing a clean, intuitive UI/UX for the ordering flow
- **What's next:** Planning improvements based on real canteen requirements

### ⚡ [Village-Renewable-Energy-Fault-Reporting-System](https://github.com/alludhanush777-gif/Village-Renewable-Energy-Fault-Reporting-System)
A platform that lets villagers easily report power/electricity breakdowns by submitting problem details, photos, and descriptions — replacing informal, delayed reporting channels.
- **Stack:** MERN (MongoDB, Express.js, React, Node.js) — built while actively learning the stack
- **Live Demo:** [village-problem.vercel.app](https://village-problem.vercel.app/)
- **Key Features:** Issue reporting form, photo upload, description submission
- **Challenge:** Getting all features fully working end-to-end

### 📄 [Reasume](https://github.com/alludhanush777-gif/Reasume)
A simple resume builder that lets users create a complete resume in one place, without needing external design tools.
- **Stack:** Next.js, React, TypeScript
- **Live Demo:** [reasume.vercel.app](https://reasume.vercel.app/)
- **Challenge:** Making the resume download/export feature work reliably
